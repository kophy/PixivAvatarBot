# PixivAvatarBot

*A Telegram Bot to get avatar from Pixiv.*    
*从p站找头像的Telegram机器人*

## 使用

在Telegram中打开[@PixivAvatarBot](https://telegram.me/PixivAvatarBot)，开始对话

目前部署在AWS上，自己运行的话，需要重命名token.conf.example为token.conf，并提供token
